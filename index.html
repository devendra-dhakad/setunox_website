How Setunox works

Privacy controls for a VPN release

VPN permission first

Android asks for VPN permission before Setunox creates a tunnel and routes device traffic through Setunox VPN servers.

Account-based access

Sign-in uses your Google account so the app can provision account access and request VPN configuration from the backend.

User deletion path

Users can delete their account in the app, or use the public account deletion page for a web-accessible request path.
